# **末梢性顔面神経麻痺について**

## **1\. 末梢性顔面神経麻痺とは？**

顔面神経は、顔の表情を作る筋肉（表情筋）を動かしたり、涙や唾液の分泌、味覚の一部などを司る神経です。末梢性顔面神経麻痺は、この顔面神経が何らかの原因で障害を受け、顔の筋肉が麻痺（動かしにくくなる）する状態を指します。

脳梗塞や脳腫瘍など、脳自体に原因がある「中枢性」とは異なり、「末梢性」は顔面神経そのものの経路に問題がある場合に起こります。

**主な原因:**

* **ベル麻痺:** 最も多く、原因不明の特発性麻痺です。単純ヘルペスウイルスの再活性化が関与していると考えられています。  
* **ラムゼイ・ハント症候群:** 水痘・帯状疱疹ウイルス（水ぼうそうや帯状疱疹の原因ウイルス）の再活性化によるもので、耳の痛みや水ぶくれ、めまい、難聴などを伴うことがあります。  
* **その他の原因:** 外傷（事故や手術）、中耳炎、耳下腺腫瘍、自己免疫疾患などが原因となることもあります。

## **2\. どのような症状が出ますか？**

症状の現れ方や程度は人によって異なりますが、一般的には以下のような症状が片側の顔面に見られます。発症後、3～7日かけて麻痺が顕在化する症例もあります

* **顔の筋肉の麻痺:**  
  * 額にしわを寄せられない  
  * 目が閉じにくい、完全に閉じられない  
  * 口角が下がり、口笛が吹けない、うがい時に水が漏れる  
  * 麻痺側の顔がゆがんで見える  
* **その他の症状:**  
  * 麻痺側の涙や唾液の分泌異常（減る、または増える）  
  * 味覚の変化（味が分かりにくい）  
  * 聴覚過敏（音が響くように感じる）  
  * 耳の後ろや顔面の痛み

## **3\. どのような検査をしますか？**

診断のため、以下のような検査が行われます。

* **神経学的検査:** 表情筋の動き、聴力、耳小骨筋反射を評価します。麻痺の程度を客観的に評価するスコアリングも行います。  
* **血液検査:** 糖尿病の程度やウイルスの関与や炎症の程度を調べるために行います。  
* **画像検査（CT、MRI）:** 経過や他の症状から脳や顔面神経の経路に腫瘍などの異常が疑われる際に行われることがあります。  
* **筋電図検査:** 神経の障害の程度や予後を予測するために発症後1−2週間程度で行われることがあります。

## **4\. どのような治療をしますか？**

治療の目的は、神経の炎症やダメージを抑え、回復を促し、後遺症を最小限にすることです。原因や重症度に応じて治療法が選択されます。

* **薬物療法:**  
  * **副腎皮質ステロイド薬:** 神経の炎症を抑えるために、発症早期（特にベル麻痺やラムゼイ・ハント症候群）に用いられます。

## **ステロイド薬の主な副作用**

| 副作用 | 詳細 |
| :---- | :---- |
| 血糖値の上昇 | 特に糖尿病をお持ちの方は、血糖値のコントロールが悪くなる可能性があります。 |
| 血圧の上昇 | 高血圧の方や、高血圧になりやすい方は、血圧が上昇する可能性があります。 |
| 不眠 | 寝つきが悪くなる、眠りが浅くなるなどの睡眠障害が現れることがあります。 |
| 消化器症状 | 胃もたれ、吐き気、食欲不振などが起こることがあります。 |
| 精神症状 | 気分が不安定になる、イライラする、興奮するなどの症状が現れることがあります。 |
| その他の副作用 | 感染症にかかりやすくなる、むくみなど。 |

* **抗ウイルス薬:** ラムゼイ・ハント症候群など、ウイルスの関与が疑われる場合に用いられます。腎臓の機能が悪い方には用量調整をします  
  * **ビタミン剤、血流改善薬:** 神経の回復を助ける目的で用いることがあります。  
* **眼の保護:** 目が閉じにくいため、角膜が乾燥したり傷ついたりするのを防ぐことが重要です。点眼薬、眼軟膏、保護用眼帯、夜間のテープ固定などを行います。  
* **リハビリテーション:**  
  * **顔面筋マッサージ:** 血行を促進し、筋肉の拘縮（こわばり）を防ぎます。  
  * **表情筋の運動:** 鏡を見ながら、ゆっくりと正しい表情筋の運動を行います。過度な運動は病的共同運動（後述）を誘発する可能性があるため、時期を見て行います。  
* **その他の治療:**  
  * **全身麻酔の手術（顔面神経減荷術）:** 重症例や特定の原因に対して検討されることがありますが、適応は限られます。

## **5\. 治りますか？（予後）**

多くの場合、適切な治療により改善が期待できます。

* **ベル麻痺:** 重症度によりますが約80%が後遺症なく治癒すると言われています。残りの方も多くは改善しますが、一部に麻痺が残ることがあります。  
* **ラムゼイ・ハント症候群:** ベル麻痺に比べて治癒率はやや低く、後遺症が残りやすい傾向があります。  
* **回復期間:** 回復には数週間から数ヶ月、場合によってはそれ以上かかることもあります。

**後遺症について:**

完全に回復しない場合、以下のような後遺症が残ることがあります。

* **顔面麻痺:** 表情筋の麻痺が残る。  
* **病的共同運動:** 口を動かすと目が閉じるなど、意図しない筋肉が一緒に動く現象。  
* **顔面拘縮:** 顔の筋肉がこわばる。  
* **ワニの涙:** 食事中に涙が出る。  
* **味覚異常、聴覚過敏**

## **6\. 早期治療の重要性**

顔面神経麻痺は、**発症後できるだけ早く治療を開始することが、回復のために非常に重要**です。
