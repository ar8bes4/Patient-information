# 📜 CHANGELOG (変更履歴)

`Patient-information` プロジェクトの変更履歴を記録します。本リポジトリはセマンティックバージョニング（SemVer）に準拠して管理されます。

## [1.0.0] - 2026-06-11

### 新規追加
- **ドキュメント整備**: プロジェクトの構成やビルド・運用方法を整理した `README.md` および本 `CHANGELOG.md` を新規作成。

### 変更・最適化
- **Git管理のクリーン化**: ビルド成果物である `HTML/` ディレクトリ配下の HTML ファイル群を Git 追跡対象から完全に除外。
  - `.gitignore` に `HTML/` を追加し、誤ったコミットを防止。
- **原本管理のSSOT化**: Markdownファイル (`src/*.md`) を唯一の正本（原本）とし、HTMLはローカルでの動的ビルドによって利用する運用へ移行。

### 初期構築（これまでの実績）
- 55件の耳鼻咽喉科・頭頸部外科疾患および手術術式のインフォームド・コンセント（IC）説明用Markdown文書の蓄積。
- 各種表示モード（画面・高齢者印刷・紙節約印刷・スライド）切り替えJSを内包したHTMLジェネレータ `generate_html.py` の安定稼働。
